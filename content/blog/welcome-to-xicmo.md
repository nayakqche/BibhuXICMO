# Welcome to Xicmo

Marketing used to mean hiring a small agency or stitching together ten tools. That is slow, expensive, and hard to measure.

We built **Xicmo** so a one-person team can run the same playbook for $99.

## One workspace, many agents

Xicmo is a single workspace that coordinates specialized agents — SEO, GEO, Reddit, X, LinkedIn, Hacker News, Content Writer, Google Search Console, Google Analytics, and a Coding Agent that opens PRs directly in your repo.

Each agent writes **action items** you approve. Nothing ships without you.

## What to do first

1. Connect your site during onboarding so we can infer industry, ICP, and voice.
2. Run the **SEO** and **GEO** agents to baseline visibility.
3. Connect one social channel and try the publish queue.

Welcome aboard.
